# Basking Ridge Community

## Run locally

```bash
npm install
npm run dev
```

## Project structure

- `index.js` runs the Express server and renders the EJS page.
- `views/index.ejs` contains the page markup.
- `public/style.css` contains the site styling.
- `public/main.js` handles menu behavior and data rendering.
- `public/data/` contains the JSON used by the browser.
- `data/source/` contains source JSON used by the helper scripts.
